# MicroGoal Management

MicroGoal Management Backend Project

## Project Description
This project is a backend API developed using Node.js and Express. 
It helps users manage their micro goals by creating, updating, and tracking daily tasks.

## Features
- Create micro goals
- Update goals
- Delete goals
- Track progress

## Technologies Used
- Node.js
- Express.js
- MongoDB
